# Task 1: 管理端配置与事件指标基础

## 描述

为管理端建立配置解析、安全配置摘要、`.env` allowlist 写入、内存事件与指标存储能力。

## 依赖

- 前置 task：无
- 外部依赖：Node.js 内置模块

## 涉及文件

| 操作 | 文件路径 | 说明 |
|------|----------|------|
| 修改 | `src/config/env.js` | 增加 `WALKER_ADMIN_ENABLED`、`WALKER_ADMIN_HOST`、`WALKER_ADMIN_PORT`、`WALKER_ADMIN_TOKEN` 解析 |
| 创建 | `src/admin/config.js` | 输出脱敏配置、可编辑字段列表、敏感字段判断 |
| 创建 | `src/admin/config-editor.js` | 只更新 allowlist 键并保留 `.env` 未知键和注释 |
| 创建 | `src/admin/event-store.js` | 保存最近 1000 条事件、指标计数、60 分钟桶统计和 session 时间线过滤 |
| 创建 | `test/admin-config-event.test.js` | 覆盖配置解析、脱敏、allowlist 写入、事件裁剪和指标统计 |

## 复杂度

medium，原因：涉及安全配置写入和内存状态模型，但不接入 HTTP 或业务服务。

## 实现步骤

- [ ] 在 `loadEnvConfig` 中解析 admin 配置，默认启用、host 为 `127.0.0.1`、port 为 `8787`、token 为空字符串。
- [ ] 在 `src/admin/config.js` 中定义可编辑环境键列表，确保 `FEISHU_APP_SECRET` 与 `WALKER_ADMIN_TOKEN` 永不作为可编辑明文字段返回。
- [ ] 在 `src/admin/config-editor.js` 中实现 `.env` 更新函数，保留注释、空行、未知键，并拒绝 allowlist 外字段。
- [ ] 在 `src/admin/event-store.js` 中实现 `recordEvent`、`listEvents`、`recordMetric`、`getMetrics`、`timelineForSession`。
- [ ] 编写 `node:test` 覆盖默认值、布尔和数字解析、脱敏、拒绝非法配置、事件上限和指标桶统计。

# Task 3: Question 回传集成兼容

## 描述

确保单选按钮和多选 v2 表单都能通过现有 `QuestionHandler` 回传 OpenCode，同时保留旧 `--toggle` 与 `--submit` 卡片兼容入口。

## 依赖

- 前置 task：T1、T2
- 外部依赖：OpenCode question reply SDK 运行时能力

| 操作 | 文件路径 | 说明 |

| 修改 | `src/dispatch/question-handler.js` | 必要时调整 `_requestOptions()`、`_sendCards()` 或 `handleAnswer()`，使 v2 表单提交继续走 `--form` 与 `_parseAnswers()`。 |
| 修改 | `test/question-handler.test.js` | 保留并补强 `--form` 多选、旧 `--toggle` 兼容、无预设题本地 TUI 降级测试。 |

medium，原因：需要在不破坏已提交按钮协议和 OpenCode question reply 链路的前提下接入 v2 表单。

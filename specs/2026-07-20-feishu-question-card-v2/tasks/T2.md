# Task 2: Card JSON 2.0 回调解析

## 描述

扩展飞书卡片事件解析，使 Card JSON 2.0 表单提交能被 Walker 归一化为现有 `/answer <key> --form <session>` 命令和 `formValue.question_selected`。

## 依赖

- 前置 task：T1
- 外部依赖：飞书 `card.action.trigger` raw v2 或 SDK 归一化事件字段

## 涉及文件

| 操作 | 文件路径 | 说明 |
|------|----------|------|
| 修改 | `src/platform/feishu/events.js` | 兼容 v2 表单提交事件中的 action/value/formValue 字段位置。 |
| 修改 | `test/feishu-events.test.js` | 增加 v2 表单提交 raw payload 和 SDK 归一化 payload 的解析测试。 |

## 复杂度

medium，原因：飞书事件字段存在 raw 与 SDK 归一化差异，解析必须兼容现有 v1 按钮回调和新的 v2 表单回调。

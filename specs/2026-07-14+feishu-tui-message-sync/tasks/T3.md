# Task 3: 双向链路集成验证与回归检查

## 描述

用独立集成测试串起已 attach session 的飞书 thread 入站与 TUI assistant 出站流程，并运行项目全量检查确认没有破坏现有行为。

- 外部依赖：无，使用内存 mock，不访问真实飞书和 OpenCode 服务

medium，原因：需要协调 dispatcher、driver watcher、消息游标和异步完成条件，但通过 mock 隔离外部服务。

- [ ] 构造已 attach 的群根 route、Walker session 和 OpenCode sessionRef。

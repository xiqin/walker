# Task 5: formValue 全链路传递

## 描述

将 `parseCardAction` 已解析的 `formValue` 从 `FeishuPlatform._handleCardAction` → `bootstrap onCardAction` → `dispatcher.handleCommand` 全链路传递，使 dispatcher 能从 `cmd.formValue` 读取表单值（multi_select/text 场景）。

## 依赖

- 前置 task：无
- 外部依赖：无

| 修改 | `src/app/bootstrap.js` | `onCardAction` 在 handleCommand 调用中传递 `formValue` |
| 修改 | `test/feishu-platform.test.js` | 新增 formValue 传递测试 |
| 修改 | `test/bootstrap.test.js` | 新增 onCardAction 传递 formValue 到 handleCommand 测试 |

low，原因：`parseCardAction` 已解析 `formValue`，只需在 3 个调用点透传该字段，无逻辑分支。

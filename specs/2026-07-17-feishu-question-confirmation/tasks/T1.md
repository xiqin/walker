# Task 1: 扩展 AgentEvent schema 支持 question metadata 和 string[] response

## 描述

扩展 `AgentEvent.DATA_SCHEMAS` 中 `EVENT_TYPE_PERMISSION` 的 metadata 字段约定，并将 `EVENT_TYPE_PERMISSION_REPLIED` 的 `response` schema 从 `string` 扩展为 `string|string[]`，为下游 question 回复（多选返回数组）提供类型契约。

- 外部依赖：无

low，原因：仅修改 schema 声明对象的字段类型字符串和注释，无逻辑分支。
